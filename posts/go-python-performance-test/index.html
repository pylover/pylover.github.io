Go & Python performance test

Vahid Mardani

Source

Consider this pseudo code:

async def worker(worker_idx, iterations):
    result, o = 0, iterations * worker_idx
    for i in range(o):
        result += i**i
    for i in range(o):
        result -= i**i

await wait([worker(i, 600) for i in range(10)])
